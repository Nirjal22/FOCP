● Method
● List comprehension
● Tuple
● Tuple Packing
● Sequence Unpacking

- Method: A function that is associated with an object in object-oriented programming.

- List comprehension: A concise way to create lists in Python using a single line of code.

- Tuple: An immutable ordered collection of elements in Python.

- Tuple Packing: The process of combining multiple values into a tuple.

- Sequence Unpacking: The process of extracting individual elements from a sequence (e.g., tuple, list) into separate variables.
